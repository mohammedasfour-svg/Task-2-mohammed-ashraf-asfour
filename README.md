# Data Classification Using AI 🤖

A beginner-friendly Artificial Intelligence project focused on building a basic classification model using Machine Learning techniques in Python.

This project was developed as part of the Artificial Intelligence Industrial Training Kit 2026 powered by DecodeLabs.

The main objective is to train an AI model to recognize patterns in data and classify new inputs based on learned information.

---

## 📌 Project Objective

The goal of this project is to:

- Load and understand a dataset
- Split the dataset into training and testing sets
- Train a machine learning classification model
- Evaluate model performance
- Understand the fundamentals of supervised learning

---

## 🚀 Features

- Dataset loading and preprocessing
- Training and testing data split
- Machine Learning classification model
- Accuracy evaluation
- Confusion matrix and classification report
- Beginner-friendly implementation

---

## 🛠 Technologies Used

- Python 3
- Scikit-learn
